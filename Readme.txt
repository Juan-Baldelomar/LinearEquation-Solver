
* compilar utilizando la herramienta make
* para ejecutar escribir $./mainApp

* Este programa resuelve sistemas de ecuaciones a traves de matrices diagonales, TS, TI y utilizando eliminación Gaussiana. También se encuentra implementada la versión de eliminación gaussiana con pivoteo completo y la factorización LU con pivoteo completo. 

* Se puede resolver un sistema de ecuaciones con la factorización LU.

* En los casos de división por 0 se lanza una advertencia, pero no se detiene la ejecución.
